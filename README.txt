A useful collection of Elementor Widgets and Extensions.

== Description ==

Revolution for Elementor is a revolutionary collection of various widgets and extensions for the Elementor Page Builder.

The free version contains the following widgets:

* Star Rating: a dynamic and versatile rating widget
* Advanced List: a dynamic and versatile list widget, similar to Elementor Pro Price List but with more features and for other use cases than just pricelists 
* Term List: a widget similarly to the Categories widget but for all (also custom) taxonomies with a lot more features
* Date Range: display different kind of dates (picker, dynamic, first post, latest post) as single date or range. Very useful for building copyright footers

The free version contains the following extensions:

* Particles: add animated particle effects to sections
* Flex Columns: configure CSS flexbox on columns using a easy-to-use, graphical interface without having to know or write CSS

The premium version adds additional widget and features

* Advanced List: display a Star Rating widget inside the Advanced List widget
* Dynamic Template: a widget which allows you to display templates from the Elementor template library by entering the ID of the template or by reading the ID dynamically via a custom field on the post
* Taxonomy Related content: display content based on the taxonomies and terms of a post. With the custom Taxonomy Related post type and the Taxonomy Related widget you can build complex websites as you were never be able before

== Installation ==

Upload through WordPress:

1. Go to your WordPress `Plugins` section
2. Click on the `Add New` button
3. Click on the `Upload Plugin` button
4. Click on the `Choose File` button
5. Select the `revolution-for-elementor.zip` file 
6. Click on the `Install Now` button
7. Activate the plugin through the 'Plugins' menu in WordPress

Manual upload: 

1. Upload `revolution-for-elementor.zip` to the `/wp-content/plugins/` directory
2. Unzip the zip file
3. Activate the plugin through the 'Plugins' menu in WordPress
